Инструкция к устанавке и запуску программы:

1. Создать директорию.
2. Скачать фалйы(game.py, game_module.py, game_module_func.py) в одну директорию.
3. Зайти через консоль в созданную ранее директорию.
4. Ввести команду "python game.py" для запуска. Команда может меняться в зависимости от версии python или ОС.
5. Для начала игры ввести "1".
6. Указать запрашиваемые параметры(имя, время задержки между ходами).
7. Для завершения сеанса ввести "2".



Требования:

1. ПК с установленным python.
